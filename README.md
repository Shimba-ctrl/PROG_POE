##  Project Overview
This project is a console-based Cybersecurity Awareness Chatbot developed in C#. The chatbot educates users about online safety, including topics such as password security, phishing attacks, and safe browsing practices.

It forms Part 1 of a Portfolio of Evidence (POE) and demonstrates basic programming concepts, user interaction, and cybersecurity awareness.

---

## Features
- Personalised user interaction (uses user’s name)
- Interactive chatbot conversation system
- Cybersecurity education (password safety, phishing, safe browsing)
- ASCII art logo and formatted console UI
- Audio greeting on startup (Windows Media Player integration)
- Input validation and error handling
- Modular structure using multiple classes

---

##  Project Structure
- `Program.cs` → Main application flow and chat loop
- `Chatbot.cs` → Handles all chatbot responses
- `UIHelper.cs` → Handles UI styling and ASCII art
- `AudioPlayer.cs` → Handles audio greeting playback
- `greeting.wav` → Audio welcome message
